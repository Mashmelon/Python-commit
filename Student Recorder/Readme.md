Overview
The Student Records Management System is a simple Python program that allows users to manage student records. 
It provides functionality to input, view, delete, and search for student data.
This system is designed to help educational institutions maintain student information efficiently.

Features
Input new student records, including name, class, section, and admission number.
View existing student records in a tabular format.
Delete student records based on name and admission number.
Search for student records by admission number or name and class.
